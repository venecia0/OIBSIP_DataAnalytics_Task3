# Data Cleaning Project

## Project Overview

This project focuses on cleaning and preparing a real-world Airbnb dataset. Data cleaning is a critical step in data analysis because it improves the accuracy, consistency, and reliability of the dataset before performing further analysis or building models.

## Dataset

**AB_NYC_2019.csv**

The dataset contains Airbnb listing information from New York City. It includes details such as host information, listing locations, room types, pricing, and availability.

## Objective

The main objective of this project is to prepare a high-quality dataset by removing inconsistencies and improving data reliability. The cleaning process ensures that the data is suitable for further analysis and visualization.

## Steps Performed

The following data cleaning steps were carried out:

1. Imported and loaded the dataset using **Pandas**
2. Examined the dataset structure and checked data types
3. Identified and handled missing values
4. Removed duplicate records from the dataset
5. Standardized categorical values to maintain consistency
6. Detected unusual values and outliers in the price column
7. Prepared a cleaned dataset ready for analysis

## Tools & Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Outcome

After performing the cleaning process, the dataset became more structured and reliable. Missing values and duplicate entries were addressed, categorical data was standardized, and outliers were reviewed to improve overall data quality.

## Conclusion

Data cleaning is an essential part of the data analysis workflow. By preparing a clean and consistent dataset, analysts can ensure that the results and insights generated from the data are accurate and meaningful.
